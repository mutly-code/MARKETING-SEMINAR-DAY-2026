Setup Guide

Follow these steps to configure Google Sheets API integration.

1 Create a Google Cloud Project

Go to Google Cloud Console and create a new project (or select an existing one).

2 Enable Google Sheets API

Navigate to APIs & Services → Library, search for Google Sheets API, and click Enable.

3 Create a Service Account

  1. Go to APIs & Services → Credentials
  2. Click Create Credentials → Service Account
  3. Give it a name (e.g., "Seminar Checkin")
  4. Click Done (no need for optional permissions)

4 Download the Credentials JSON

  1. Click on the service account you just created
  2. Go to the Keys tab
  3. Click Add Key → Create New Key → JSON
  4. Rename the downloaded file to credentials.json
  5. Move it to: server/credentials.json

5 Set Up the Google Sheet

Create a new Google Sheet and add these exact column headers in row 1:

ID | Name | Email | Role | Table Number | Dietary Restrictions | Check-in Status | Check-in Time | QR Code URL

Then share the sheet with your service account email (found in the JSON file under client_email). Give it Editor access.

6 Configure Environment Variables

Copy the Sheet ID from the URL of your Google Sheet:

https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID_HERE/edit

Then update the .env file in the project root:

GOOGLE_SHEET_ID=your_sheet_id_here
